Getting Started

Heres a quick guide to using the software chat index for the first time.

First, make sure that you have not messed with the software too much so that it works.

Then to open a new chat page, click the New Tab button (pluse icon on the right side of the tab bar)
or double-click the tab bar. If this worked you should be in a new tab.

Next, click inside the input box to type your message.
Make sure you click somewhere that is not on the first line
(reserved for file uploads) or the last line (reserved for the toolbar).
if it is unclear then see that when the mouse is hovering over the textbox then
the mouse style is the text cursor.

Type your question or message into the chat box.
Then press Enter (or Return on some computers)
or click the Send button in the lower-right corner of sayed input box
to submit it to the AI. Of if you died something wrong do nothing. (and the error is in the console)
If you want to create a multi-line message, use Shift + Enter to insert a new line into the message.

if you got a responce then, congratulations!
You now know how to open new tabs and send queries.
